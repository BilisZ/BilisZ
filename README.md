## 🔧 What I Do

- **Networking & Hybrid Connectivity** — Transit Gateway, Site-to-Site VPN, Client VPN, Direct Connect, cross-cloud VPN (AWS ↔ Azure), DNS resolution, AWS Network Firewall
- **Kubernetes & Containers** — EKS cluster operations, upgrades, networking troubleshooting, aws-load-balancer-controller, ECS scheduling
- **Security & Incident Response** — WAF, GuardDuty, PaloAlto firewall management, CVE remediation, SIEM integration (Google Chronicle, Microsoft Sentinel)
- **Cost Optimization** — Billing analysis, rightsizing, data transfer optimization, cost anomaly investigation
- **Database Operations** — RDS (PostgreSQL, MySQL, MSSQL), ElastiCache/Redis, Redshift, Elasticsearch upgrades and performance tuning
- **Monitoring & Observability** — CloudWatch alarms/queries, account health reviews, centralized logging
